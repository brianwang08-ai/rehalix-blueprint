# Rehalix Interactive Site Blueprint

A public, zero-login interactive blueprint for the Rehalix independent website.

## Live site

https://brianwang08-ai.github.io/rehalix-blueprint/

## What it includes

- Clickable site architecture and page relationships
- Buyer journeys for distributors, institutions, OEM/ODM buyers, and professional researchers
- Page-level wireframes
- Content strategy and capability system
- Open decision register

The site uses no external data source and is deployed automatically through GitHub Pages.
